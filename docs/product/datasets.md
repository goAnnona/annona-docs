# Datasets

Datasets represent the information available to your Annona tenant.

## Typical dataset tasks

In the customer product, you should be able to:

- review the datasets available to your organization
- inspect dataset versions and freshness
- understand which uploads created which dataset versions
- follow links from datasets into the related agent or trace flows

## Isolation expectation

Your dataset experience should be scoped to your tenant only. You should not be able to see other customers' datasets or lineage.
