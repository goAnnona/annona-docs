# Agents

Your Annona tenant may include more than one agent experience, depending on your configuration.

## Grounded and generic answers

Annona may show both a more generic answer path and a grounded Annona answer path. The grounded path should be explicit about what data it used and what evidence supports the answer.

## What customers should expect

Customers should be able to:

- ask questions against their own tenant's data
- understand when an answer is grounded in uploaded datasets
- review evidence and trace context when needed
- stay inside their own tenant boundary

Customers should not be routed into internal admin testing screens.
