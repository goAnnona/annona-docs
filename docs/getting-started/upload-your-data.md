# Upload your data

Uploading data is the first step in setting up many Annona tenants.

This page remains available for older links. New in-product upload help should target [Uploads](../uploads.md).

## What to expect

A typical upload flow should help you:

1. choose your target dataset or upload area
2. select a file or bundle
3. review validation feedback
4. submit the upload
5. wait for readiness and processing to finish
6. open the related dataset, agent, or trace view

## After upload

Once your upload is processed, Annona should make it easy to:

- see whether the upload completed successfully
- understand what dataset version was created
- open the relevant dataset detail
- ask your agents questions against the uploaded data
- inspect your own trace history for that activity

## Need help inside the app?

If the upload experience includes help text or help buttons, those links should land on customer docs pages in this site.
