# Annona Docs

MkDocs-based documentation hub for Annona.

## Local preview

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

## Initial scope

- delivery proofs and what is truly working end to end
- operator and developer guides
- ingestion architecture and readiness documentation
- runbooks for DEV-only deploy, smoke, and proof flows
