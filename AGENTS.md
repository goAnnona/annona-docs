# AGENTS.md

## Repo purpose

This repo is the canonical documentation hub for Annona.

Use it for:
- product and architecture docs
- end-to-end proof and operator guides
- deployment and runbook documentation
- ingestion and tracing documentation

## Working rules

- Treat docs as a first-class product surface, not an afterthought.
- Prefer updating canonical docs in the same PR as behavior changes.
- Be explicit about current live status versus planned behavior.
- If something is not live end to end, say so plainly.
- Screenshots and proof artifacts should be tied to real validated states, not mockups unless labeled.
